Intrucciones:
1) Al ejecutar el archivo .py va a aparecer un menú de 3 opciones ( 1-Modo Consola, 2-Consola + Graficos o 3-Solo Graficos)
2) Digitamos el número del modo que queramos ejecutar

Modo Consola:
El modo consola no trae el tablero solo se juega mediante consola y constantemente se imprimen las casillas correspondientes de cada ficha.

Modo Consola + Graficos:
Este modo es 100% jugable por consola incluye una interfaz con el tablero ,unos dados que cambian según lo 
que saque el jugador, y al pasar el mouse por una ficha o casilla saldra su nombre o número en un recuadro
naranja.

Modo Graficos:
Este modo es 100% grafico, las entradas se reciben por medio de una caja de texto, y un dropdown menu o combo box, para elegir un dado hay que
pulsarlo, y para elegir una jugada se elige en el menu, dando click encima se despliegan las posibilidades, cuando aparezca una caja de texto se 
borra el texto que haya y se pone el requerido( Explicacion a fondo en los ultimos minutos del video)

Contraseña Modo Desarrollador:
Para iniciar el juego en modo desarrollador se debera ingresar la contraseña: "mazamorra" , cuando el juego lo pida
